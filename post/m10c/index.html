組態

在config.toml文件中，在中定義以下變量params：

  • author：作者姓名
  • description：作者簡介
  • avatar：包含作者頭像圖像的文件的路徑
  • menu_item_separator：每個菜單項之間的分隔符。允許HTML（默認：“-”）

要添加菜單項，請在中添加以下行menu：

[[menu.main]]
  identifier = "tags"
  name = "Tags"
  url = "/tags/"

閱讀Hugo文檔以獲取有關菜單的更多信息

要添加社交鏈接，請在中添加以下行params：

[[params.social]]
  name = "github"
  url = "https://github.com/vaga"

要更改主題顏色，請在中添加以下行params：

[params.style]
  darkestColor = "#d35050"
  darkColor = "#212121"
  lightColor = "#f5e3e0"
  lightestColor = "#f5f5f5"
  primaryColor = "#fff"

如果需要以上主題顏色，則可以查看exampleSite / config.toml文件。

造型

要使用SCSS覆蓋樣式，添加一個名為_extra.scss來{your site directory}/assets/css/

注意：Hugo版本有兩個版本，hugo和hugo_extended。您需要hugo_extended自動編譯您的scss。
